setup: 
	INSTALL LIBMAGIC
	g++ -o file-org file-org.cpp -lmagic

How to use: 
	when prompted, enter the path to the .csv file you want to use for 
	the settings. (the default settings can be found in the folder 
	SETTINGS).
	Then enter the directory path to the directory that you want to 
	organise the files within it.
